Detailed Description

Заголовочный файл со SFINAE перегрузками функции print_ip для различных типов

Данный файл содержит перегрузки функции print_ip для таких типов, как std::string, целочисленные типы, std::tuple (при условии, что все члены одного типа), А так же контейнеры std::list и std::vector
